# גל-מערכות רכב (1992) בע"מ - אתר תדמית

דף נחיתה (React + Vite) בעברית/RTL, כולל עמוד **מדיניות פרטיות** הנדרש לאימות עסק בפייסבוק.

## מבנה התיקייה

```
gal-car-systems/
├── index.html            # נקודת כניסה + תגי SEO/Open Graph
├── package.json
├── vite.config.js
├── public/
│   └── logo.webp         # הלוגו של החברה
└── src/
    ├── main.jsx
    ├── App.jsx           # כל תוכן הדף + מדיניות הפרטיות
    └── index.css         # פונטים, צבעי מותג, RTL
```

## הרצה מקומית

צריך Node.js 18 ומעלה.

```bash
npm install
npm run dev
```

ייפתח בכתובת http://localhost:5173

## בנייה לפרסום

```bash
npm run build      # מייצר תיקיית dist/
npm run preview    # תצוגה מקדימה של גרסת הפרודקשן
```

## העלאה לאוויר (כדי לקבל כתובת URL לפייסבוק)

הדרך הקלה ביותר - **Vercel** או **Netlify**:

1. פתחו חשבון (חינם) ב-vercel.com או netlify.com.
2. העלו את התיקייה (גרירה) או חברו ל-GitHub.
3. הגדרות בנייה: Build command = `npm run build`, Output directory = `dist`.
4. תקבלו כתובת קבועה, למשל `https://gal-car-systems.vercel.app`.

לאימות העסק בפייסבוק תוכלו להשתמש בכתובת הראשית, וקישור "מדיניות פרטיות" נגיש מהתפריט ומהפוטר.

## לפני העלאה - חובה למלא

חפשו בקובץ `src/App.jsx` את הסימון `[הזינו` והחליפו ב:

- מספר טלפון
- כתובת דוא"ל
- כתובת העסק

(מופיעים גם בסעיף "צור קשר" וגם במדיניות הפרטיות, סעיף 10.)

## פיקסל פייסבוק (אופציונלי)

אם תתקינו Meta Pixel, סעיף 5 במדיניות הפרטיות כבר מכסה זאת. את קוד הפיקסל עצמו מוסיפים ב-`index.html` לפי ההוראות מ-Meta Events Manager.
